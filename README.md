HelloBoost. C++ Tutorials. Boost Library. Embarcadero Rad Studio IDE. Simple Input/Output. Console Application. 

The program does the following:
  1. Asks the user for one or more numbers in a space-separated string.
  2. Multiplies it (them) by 3 and displays the result on the screen, either one or several in a space-separated line.
     A lambda expression is used for multiplication.
  3. Repeats the process until the key combination "ctrl-c" or "ctrl-z" is pressed.

HelloBoost. Занятия по C++. Библиотека Boost. ИС Embarcadero Rad Studio. Простой ввод/вывод. Консольное приложения. 

Программа делает следующее:
  1. Запрашивает у пользователя числа либо одно либо несколько в строку через пробел.
  2. Умножает его (их) на 3 и выводит резльтат на экран либо одно либо несколько в строку через пробел.
     Для умножения используется лямбда-врожение.
  3. Повторяет процесс до тех пор пока не будет нажата комбинация клавиш "ctrl-c" или "ctrl-z".
